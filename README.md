# igname-distribution-optimization
# LRM-0: Local Resource Logic  Optimisation des flux thermiques et gestion d'inertie en milieu décentralisé (contexte igname).   **Note système :** L'optimisation s'arrête au seuil d'imprévisibilité du flux d'entrée. Ne pas réduire au-delà du bruit thermique. Si perte de Verbe, retour immédiat à l'état source (cuisine). Le réel est irréductible.
